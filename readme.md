# MonCommerce

MonCommerce est un proet d'apprentissage.

## Environnement de développement

### Pré-requis

* Symfony CLI
* Composer
* PHP >= 7.4
* Yarn & Nodejs

### Lancement de l'environnement

```bash
symfony serve -d
```

### Fixtures

```bash
symfony console doctrine:fixtures:load -n
```
